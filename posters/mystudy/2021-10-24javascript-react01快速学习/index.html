0、jsx初识

what — jsx是js的语法扩展，可以很好的描述UI应该呈现出它应有的交互本质形式(有些像模版语言)，jsx可以生成React元素.

Why — React认为渲染逻辑本质上与其他UI逻辑内在耦合，比如：UI中需要帮定处理事件、在某些时刻状态发生改变时候需要通知到UI，以及需要在UI中展示准备好的数据。

React并没有采用将标记与逻辑分离到不同文件的这种人为的分离方式，而是通过将二者共同放在称之为"组件"的松散耦合单元之中，来实现关注点分离。

how – jsx是一个表达式，表达式中可以嵌入表达式、也可以指定属性、指定多个子元素

function getGreeting(user) {
  if (user) {
    return <h1>Hello, {formatName(user)}!</h1>;
  }
  return <h1>Hello, Stranger.</h1>;
}

const element = (
  <div>
    <h1>Hello!</h1>
    <h2>Good to see you here.</h2>
  </div>
);

jsx表示一个对象，如下：

const element = (
  <h1 className="greeting">
    Hello, world!
  </h1>
);

等价于：

const element = React.createElement(
  'h1',
  {className: 'greeting'},
  'Hello, world!'
);

的原始代码为：

// 注意：这是简化过的结构
const element = {
  type: 'h1',
  props: {
    className: 'greeting',
    children: 'Hello, world!'
  }
};

1、jsx语法

React核心包 //解析jsx语法，并转义虚拟dom对象

React-dom核心包 //将虚拟dom的对象渲染到页面

render参数：（1）jsx元素（2）挂载点根节点只能有一个div（3）回调

jsx：

（1）javascript+xml在javascript里面嵌入了html 但是这个内容不是字符串,而是一个jsx对象

（2）根部只能有一个div

（3）{}用来引用变量

必须有返回值

不可以是对象(必须带属性)，如果内容是css样式，则可以通过css的编译

可以是元素非对象的数组

基本的数据类型可以：null，boolean，undefined 显示为空不报错

支持运算、字符串链接符

不支持if else，可以三元运算

支持映射

在jsx语法里面无法使用class，class是es6的保留关键字

let name = {

2、jsx编译流程

（1）javascript 编译工具，babel-preset-react将jsx元素转为React.reateElement()来处理

（2）React.createElement 转为一个虚拟dom对象，格式如下：

{
		type:div
		props:{
				id:"",
				className:"",
				style:"",
				children:""
		},
		key:"",
		ref:"",
		
}

（3）手工实现dom渲染

//==========	functionAll.js	=============
export function createElement(type, props, children) {
  
  //　如何type为一个函数，返回一个函数执行后值
    let obj = {
        type:typeof type === 'function'?type():type,
        props: {...props, children},
        key: props.key || undefined,
        ref: props.ref || undefined
    }
    return obj;

}

export function render(obj, container, callback) {
    let {type, props} = obj;
    let {children} = props;


    //创建一个元素
    let el = document.createElement(type);

    for (let arr in props){
        let value = props[arr];
        //第一种处理className
        //第二种定义样式
        //第三种将普通属性赋到元素上面

        switch (arr){
            case 'className':
                el.setAttribute('class',value);
                break;
            case 'style':
                for(let key in value){
                    el['style'][key] = value[key]
                }
                break;
            default:
                el.setAttribute(arr,value)
                break;
        }
    }

    //创建一个文本节点，children为文本的时候
    let textNode = document.createTextNode(children);
    el.appendChild(textNode);

    callback && callback();

    //将手动的创建的el挂载到容器中
    document.getElementById(container).appendChild(el)

}



//==========	index.js	=============
import {createElement, render} from "./functionAll";

let obj = createElement(
    'div',
    {
        id: 'one', ref: 'aaa', key: 22, className: 'one', style: {color: red}
    },
    'abcdefg'
)


render(obj, 'root', function () {
    console.log("成功!!!")
});
