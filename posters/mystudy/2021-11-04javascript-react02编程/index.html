2021-11-07javascript-react快速入门

函数组件

1、两个组件数据传递，通过props来传递修改（props可以包含从父组件的属性和方法）

(1)第一步在父组件中定义相应的属性**(定义在this.state = {}中)**、方法

属性通过this.state或方法通过this.fun传给引入子组件的属性标签，如下：

//子组件
import React,{Component} from "react";
class Dialog extends Component{
    constructor() {
        super();
    }

    render() {
        return(
            <div className="mask" style={{display:this.props.display}}>
                <div className="content">
                    <button onClick={this.props.hide}>关闭</button>
                    {this.props.title}
                </div>
            </div>
        )
    }

}

export default Dialog;

//父组件
class Statisc extends Component {

    constructor() {
        super();
        this.state = {
            display:"none"
        }
    }

    render() {
        let props = this.props
        return (
            <div>
                <Navbar/>

                <Layout className="layout">

                    <div className="site-layout-content">
                        <h3>统计</h3>
                        <h5>{props.name}</h5>
                        <di>
                            <button onClick={this.tan}>弹出</button>
                            <Dialog display={this.state.display} hide={this.hide} title='这是一个弹框！！'/>
                        </di>
                    </div>

                    <MyFooter/>
                </Layout>
            </div>
        );
    }

    tan = () => {
        this.setState({display: "block"});
    }

    hide = () =>{
        this.setState({display:"none"})
    }
}

(2)子组件接受属性（通过this.props.xxx）来更改样式、或调用方法

注意props.xxx的值是单向不可改变的（只读的），如果改变可以在回调方法中操作

props是用来父传递子组件的中间变量、控制外部数据，state来控制内部数据

(3)特别强调方法的传递，在子组件产生一个事件，这个事件指向组件属性（也是通过this.props.xxx），而这个组件的属性指向父组件的方法

(4)父组件的这个方法（其实一个是异步回调）通过改变状态来更改属性状态来更改变子组件中属性属性状态，数据单项绑定

ref用法（https://segmentfault.com/a/1190000008665915）

Why - 在react典型的数据流中，props传递是父子组件交互的唯一方式；通过传递一个新的props值来使子组件重新re-render,从而达到父子组件通信。在react典型的数据量之外，某些情况下（例如和第三方的dom库整合，或者某个dom元素focus等）为了修改子组件我们可能需要另一种方式，这就是ref方式。

What - React提供的这个ref属性，表示为对组件真正实例的引用，其实就是ReactDOM.render()返回的组件实例；需要区分一下，ReactDOM.render()渲染组件时返回的是组件实例；而渲染dom元素时，返回是具体的dom节点。


    const domCom = <button type="button">button</button>;
    const refDom = ReactDOM.render(domCom，container);
    //ConfirmPass的组件内容省略
    const refCom = ReactDOM.render(<ConfirmPass/>,container);
    console.log(refDom);
    console.log(refCom);

挂到组件（这里组件指的是有状态组件）上的ref表示对组件实例的引用，而挂载到dom元素上时表示具体的dom元素节点。

How -

ref可以设置回调函数
ref可以设置字符串
获取ref引用组件对应的dom节点
